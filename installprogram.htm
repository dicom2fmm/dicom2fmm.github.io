Установка программы

 Top  Previous  Next

Программа поставляется ввиде установочного  файла

 

•dicom2fmm.msi (для 32/64-разрядной системы, щёлкните на ссылку для загрузки).

 

Для работы программы необходимо установить Access Database драйвер -

AccessDatabaseEngine.exe (щёлкните на ссылку для загрузки).

а также среда выполнения .NET Framework - dotNetFx40_Client_x86_x64.exe (щёлкните на ссылку для загрузки).

 

Для установки программы достаточно запустить установочный файл (dicom2fmm.msi,

щёлкните на ссылку для загрузки) с указанием "клиентской" папки.

На пример, P:\Clients\WorkingVersions\dicom2fmm\

  В базе данных хранятся все сведения о пациенте, о процессе лечения, а также ссылки на физические объекты,

такие, как томограммы (отснятые на установке и сторонние), объекты DICOM-RT (томограммы с оконтуренными зонами облучения),

созданные планы облучения, файлы с историей процесса лечения, фотографии пациента.

Физически все эти объекты располагаются в личной папке пациента по пути P:\P_DATA\ФИО_ID

Здесь ФИО – фамилия, имя, отчество пациента подряд без пробелов,

ID – идентификационный номер пациента в базе данных.

 

Пример: P:\P_DATA\СертификатовФантомФантомович_302

 

Папка и все необходимые подпапки создаются в программе «Автоматизированная история болезни»

автоматически при регистрации пациента.

 

Необходимо создать источники данных ODBC:

BasePTC  - для 32-х разрядных версий приложений

BasePTC64 - для 64-х разрядных версий приложений

 

Проверить, установлены ли источники данных, можно по ветке

Панель управления -> Администрирование -> Источники данных ODBC.

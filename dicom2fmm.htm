Создание FMM на основе DICOM

 Top  Previous  Next

Процесс создания FMM файла на основе DICOM томограммы состоит из следующих этапов:

1.Загрузка DICOM томограммы с контурами, полученной после программы оконтуривания.

2.Редактирование контуров.

3.Создание FMM файла на основе DICOM томограммы с контурами, используя один из выбранных
алгоритмов преобразования единиц Хаунсфилда (HU) в единицы FMM .

4.Завершение программы с записью результата в Базу данных
